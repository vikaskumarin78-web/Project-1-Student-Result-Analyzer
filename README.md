This project is a simple yet effective tool designed to automate the process of academic grading. By following a structured logic flow, the program ensures accurate calculations and a professional presentation of results.

How it Works
The application follows these core execution steps:

Ask name: The program starts by prompting the user to enter the student's name for a personalized report.

Use loop for 5 subjects: Instead of redundant code, a loop is utilized to efficiently collect marks for five different subjects.

Calculate total: Once the marks are gathered, the system aggregates them to find the overall score.

Calculate average: The total is divided by the number of subjects to determine the mean percentage.

Use if-else for grade: Based on the calculated average, conditional logic assigns a specific letter grade (e.g., A, B, C).

Print final result cleanly: All data is compiled into a well-formatted summary, displaying the name, total, average, and final grade.
